  • Watch the Pattern: A random color will flash on the screen. Watch carefully as the game shows the sequence.
  • Repeat the Sequence: Click the colors in the same order as shown. Each level adds one more color.
  • Level Up: Complete the sequence correctly to move to the next level with a longer pattern.
  • Be Careful: If you click the wrong color, the game ends. Restart to try again.
  • Goal: Test your memory and see how many levels you can complete!
Example:
Suppose the game flashes Red, then Green. You need to click Red first, then Green to complete the level. On the next level, the sequence may become Red, Green, Yellow, and you will repeat this exact order.
